# Application Security
### Web Browser Security (OBJ 3.2 & 3.3)
Ensure you web browser is up-to-date with patches…
But don’t adopt the newest browser immediately

Which web browser should I use?
Depends on OS
Chrome and Firefox are pretty much universal

General Security for Web Browsers

- &nbsp;
		1. Implement Policies
			1. Create and implement web browsing policies as an administrative control or technical control
		1. Train your Users
			1. User training will prevent many issues inside your organization
		1. Use Proxy & Content Filter
			1. Proxies cache the website to reduce requests and bandwidth usage
			2. Content filters can be used to blacklist specific websites or entire categories of sites
		1. Prevent Malicious code
			1. Configure your browsers to prevent ActiveX controls, Java applets, JavaScript, Flash, and other active content

### Web Browser Concerns (OBJ 3.2 & 3.3)
		Cookies
			Text files placed on a client’s computer to store information about the user’s browsing habits, credentials, and other data

			Tracking cookie - Spyware
			Session Cookie - User preferences

		Locally Shared Object (LSO)
			Also known as flash cookies, they are stored in your Windows user profile under the Flash folder inside of your AppData folder

		Add-ons
			Smaller browser extensions and plugins that provide additional functionality to the browser

		Advanced Security Options
			Browser configuration and settings for numerous options such as SSL/TLS settings, local storage/cache size, browsing history, and much more

### Securing Applications (OBJ 1.4 & 3.2)

		Microsoft Office
			Use passwords to protect the contents of your documents

		Macros
			Check Mac settings / disable by default

		Digital Certificates
			Only let people with digital certificates to open documents

		Digital Signatures and digital certificates are used by MS Outlook for email security

		Password protect email archives

		User Account Control
			Prevents unauthorized access and avoids user error in the form of accidental changes
