# Perimeter Security
Security devices focused on the boundary between the LAN and the WAN in your organization’s network
Perimeter security riles on several different devices

### Firewalls
Firewalls screen traffic between two portions of a network
Software
Software running on a host or server
Hardware
Looks like a switch
Embedded

Packet Filtering
Inspects each packet passing through the firewall and accepts or rejects based on the rules
Stateless Packet filtering
Accept or reset based on IP and port number
Stateful packet filtering
tracks the requests leaving the network
NAT filtering
Filters traffic based upon the ports being utilized and type of connection (TCP or UDP) ‘
Application Layer gateway
Conducts an in-depth inspection based upon the application being used
Circuit-Level gateway
Operates at the session layer and only inspects the traffic during the establishment of the initial session over TCP or UDP
MAC Filtering
Filtering devices via MAC address
Explicit Allow
Traffic is allowed to enter or leave the network because there is an ACL full that specifically allows it.
