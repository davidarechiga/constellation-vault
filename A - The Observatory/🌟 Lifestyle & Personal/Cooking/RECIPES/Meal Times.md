# Meal Times & Categories

Meal time and recipe category are **tags**, not folders. Add the relevant tags to any recipe's frontmatter.

## Meal Time Tags

| Tag | When to use |
|-----|-------------|
| `#breakfast` | Morning meals — eggs, toast, oatmeal, smoothies |
| `#brunch` | Late morning / weekend leisurely meals |
| `#lunch` | Midday meals — sandwiches, salads, soups, lighter fare |
| `#dinner` | Evening meals — mains, sides, full plates |
| `#snack` | Small bites between meals |
| `#mealprep` | Recipes designed to batch cook and store |

## Category Tags

| Tag | When to use |
|-----|-------------|
| `#dessert` | Sweet treats and post-meal courses |
| `#ice-cream` | Ice cream and frozen desserts |
| `#mocktail` | Non-alcoholic cocktail-style drinks |
| `#drinks` | Juices, tonics, smoothies |
| `#sauce` | Salsas, marinades, condiments |
| `#sick` | Remedies and comfort food for illness recovery |

A recipe can have **multiple tags** — e.g. a frittata is both `#breakfast` and `#brunch`. Use Obsidian's tag search or filter in a `.base` view to browse by any combination.
