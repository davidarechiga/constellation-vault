## 🧠 What Is This?

This is **Constellation** - a personal knowledge management system built in Obsidian. It's designed to capture, organize, and connect everything important in your life: projects, ideas, poetry, learning, relationships, and career development.

**Core Philosophy:**
- **Capture everything** worth remembering
- **Organize** using the PARA method
- **Connect** ideas through wikilinks
- **Review** regularly to maintain and grow

### Hub Files: Your Navigation Centers

**What are Hubs?**
Hub files are your navigation centers - they connect related content across Projects, Areas, and Resources.

**How to Use Hubs:**
1. Start at a hub (e.g., [[Music Ecosystem Hub]])
2. Browse the organized sections
3. Click wikilinks to jump to specific notes
4. Use the "Recent Activity" dataview to see latest work
5. Follow "Related Hubs" to explore connected topics

**All Hubs:**

## 📅 Daily Workflows

### Morning Routine

1. **Open Home Dashboard**
   - Click `📔 Today's Daily Note` (auto-dated link)
   - If daily note doesn't exist, it creates from template

2. **Review "Tonight's Prep" Section**
   - Check what you prepared last night
   - Adjust if needed

3. **Plan Your Day**
   - Fill in Schedule section (Morning/Afternoon/Evening)
   - Review Priority Tasks on Home dashboard
   - Set your #1 focus

4. **Start Working**
   - Navigate via Hubs to relevant content
   - Create project notes as needed

---

### Evening Routine

1. **Open Tomorrow's Daily Note**
   - Click the daily note link
   - Fill out "Tonight's Prep for Tomorrow" section
   - Plan breakfast, lay out clothes, review calendar

2. **Complete Today's Recap**
   - Mark habits (use `1` for done, blank for not done)
   - Log energy check-ins
   - Write reflection: Wins, Improvements, Gratitude

3. **Review Tomorrow**
   - Check calendar for appointments
   - Identify tomorrow's #1 priority
   - Write 3 intentions

---

### Weekly Review

**When:** Sunday evenings or Monday mornings

1. **Create Weekly Review Note**
   - Use `Templates/Weekly Review Template.md`
   - Date it for the week you're reviewing

2. **Review Process:**
   - Wins & accomplishments
   - Habit consistency (automatic dataview)
   - Project progress for all active projects
   - Creative output
   - Relationship check-in
   - Career development
   - Energy & wellness

3. **Plan Ahead:**
   - Set top 3 priorities for next week
   - Review scheduled events
   - Set weekly intention

## ✍️ Creating Notes

### Where Should This Note Go?

**Ask yourself:**

1. **Does it have a deadline?** → Projects
   - "Release album by March 2026" → Projects

2. **Is it ongoing responsibility?** → Areas
   - "Weekly meal planning" → Areas/Cooking
   - "Music practice techniques" → Areas/Music Performance

3. **Is it reference/knowledge?** → Resources
   - "Poetry I wrote" → Resources/Poetry
   - "Design principles" → Resources/Design
   - "Book notes" → Resources/Book Notes

4. **Is it completed/inactive?** → Archive
   - "2023 Wedding planning" → Archive

---

### Using Wikilinks

Obsidian uses **wikilinks** to connect notes:

**Basic wikilink:**
```markdown
[[Note Title]]
```

**Wikilink with custom display:**
```markdown
[[2 Music Release (In Progress)|Current Album]]
```

**Linking to headers:**
```markdown
[[Career Development Hub#Design Career Transition]]
```

**🔑 Pro Tip:** Wikilinks work by note title, not file path. Moving files doesn't break links!

## 🔍 Search & Discovery

### Obsidian Search Features

1. **Quick Switcher** (Cmd/Ctrl + O)
   - Type note name to jump instantly
   - Fuzzy matching (type "mueco" finds "Music Ecosystem")

2. **Global Search** (Cmd/Ctrl + Shift + F)
   - Full-text search across all notes
   - Supports regex

3. **Omnisearch Plugin**
   - Enhanced semantic search
   - Searches across all content

4. **Graph View**
   - Visual representation of note connections
   - Shows wikilink relationships

5. **Backlinks Panel**
   - Shows all notes linking to current note
   - Helps discover connections

## 💡 Best Practices

### ✅ Do:
- **Start from Home** - Use the dashboard as your launchpad
- **Use Hubs** - Navigate via hubs, not folder browsing
- **Link liberally** - Create wikilinks to connect ideas
- **Capture immediately** - Write it down before you forget
- **Daily notes** - Build the habit of daily logging
- **Review weekly** - Maintain the system with weekly reviews
- **Trust the system** - Information is findable when organized by PARA

### ❌ Don't:
- **Don't over-organize** - PARA is enough structure
- **Don't hoard** - Archive or delete truly obsolete content
- **Don't duplicate** - Search before creating
- **Don't orphan notes** - Link new notes from at least one place
- **Don't skip reviews** - System decays without maintenance
- **Don't overthink** - When in doubt, put it in Inbox and organize later

## 🆘 Troubleshooting

### "I can't find a note"

1. **Try Quick Switcher** (Cmd/Ctrl + O) - Type partial name
2. **Use Omnisearch** - Semantic search
3. **Check relevant Hub** - Browse organized sections
4. **Global Search** - Full-text search for keywords
5. **Check All Orphaned Notes database** - Unlinked notes

### "Where should this go?"

**Decision tree:**
1. Deadline? → Projects
2. Ongoing? → Areas
3. Reference? → Resources
4. Done? → Archive
5. Unsure? → Inbox (organize later)

### "Too many notes in Misc & Unsorted"

**Batch process:**
1. Set aside 30 minutes
2. Review 20-30 files at a time
3. Ask: "What category is this?"
4. Move to appropriate folder
5. Repeat weekly until cleared

### "Links broke after moving files"

**Wikilinks are resilient** - they resolve by title, not path.
- **If broken:** Note might be renamed or deleted
- **Check:** Use "Update internal links" when moving in Obsidian
- **Verify:** Search for the note title to confirm it exists
