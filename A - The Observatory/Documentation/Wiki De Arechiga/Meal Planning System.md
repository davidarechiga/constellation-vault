# Meal Planning System

## Overview

_How we plan meals for the week — saves money, reduces waste, and avoids the daily "what's for dinner?" stress._

---

## Weekly Meal Planning Process

1. Check what's already in the fridge/pantry
2. Pick 5-6 dinners for the week (leave 1-2 nights for leftovers/takeout)
3. Build grocery list from recipes
4. Shop once (ideally on the same day each week)
