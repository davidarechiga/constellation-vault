# Seasonal Home Maintenance Checklist

## Overview

_Quarterly home maintenance tasks to keep the house running smoothly and prevent costly repairs._

---

## Spring

- [ ] Deep clean the whole house
- [ ] Check smoke detectors / replace batteries
- [ ] HVAC filter replacement
- [ ] Clean windows
- [ ] Check weather stripping on doors/windows

## Summer

- [ ] Check and clean fans
- [ ] Inspect outdoor areas
- [ ] Declutter and donate unused items
- [ ] Check fire extinguisher

## Fall

- [ ] HVAC filter replacement
- [ ] Check smoke detectors / replace batteries
- [ ] Weatherproof windows if needed
- [ ] Stock emergency supplies for winter storms

## Winter

- [ ] Check pipe insulation
- [ ] Test heating system
- [ ] Stock up on essentials in case of storms
- [ ] Year-end document review (see [[Important Documents Locations]])

---

## Tips & Notes

- Set quarterly calendar reminders for these checks
- Some tasks depend on whether renting or owning — coordinate with landlord if renting
