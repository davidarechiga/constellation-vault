# Provisioning Steps

**Approval:** Hiring Manager 

**Provisioning:** IT Helpdesk

1. Sign in to the [OneLogin Admin console](https://masterclass.onelogin.com/admin2) >> Users >> Roles >> “Adobe Creative Cloud [Glue API]"
2. Select Users and search for the requester. Select their name and click **Check**
3. Select **Add to Role** and then click **Save** in the upper right.
4. The user will now have access to Asana via their OneLogin portal.

5. Sign in to the [Adobe Enterprise Admin Console](https://adminconsole.adobe.com/07CD5BAA5F2B38840A495C6D@AdobeOrg/overview)
	- Verify you are in the Adobe Enterprise Admin Console by selecting “MasterClass” in the upper right of the console and select “Masterclass” with a lowercase “c”
1. Select the “Products” tab in the top menubar
	- Select Creative Cloud
		- Note: Acrobat Pro DC is included in Creative Cloud so if the request is for Acrobat AND any other app, only Creative Cloud is needed.
1. Select the “Users” tab and search for an existing account.
1. If the user does nott have an existing account, select “Add User”
