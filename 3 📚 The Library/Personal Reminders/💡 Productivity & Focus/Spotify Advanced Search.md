<span style="color:#000ff;">Spotify Advanced Search</span>

<span style="color:#000ff;">You can refine your searches in Spotify by adding extra terms into the search box. Note: search terms are not case sensitive.</span>
<span style="color:#000ff;">Search by artist</span>
- <span style="color:#000ff;">artist:Radiohead –</span> <span style="color:#000ff;"><i>lists all tracks by Radiohead</i></span>
<span style="color:#000ff;">Search by track</span>
- <span style="color:#000ff;">track:"My Sound" –</span> <span style="color:#000ff;"><i>lists tracks with the name My Sound</i></span>
<span style="color:#000ff;">Search by album</span>
- <span style="color:#000ff;">album:Loveless –</span> <span style="color:#000ff;"><i>lists the album Loveless</i></span>
- <span style="color:#000ff;">album:”Many Faces Out Of Focus” –</span> <span style="color:#000ff;"><i>use quotes when the keyword contains spaces</i></span>
<span style="color:#000ff;">Search by year</span>
- <span style="color:#000ff;">year:1969</span>
- <span style="color:#000ff;">year:1989-2013</span>
- <span style="color:#000ff;">year:1989-2013 NOT year:1993</span>
<span style="color:#000ff;">Refine your search with AND, OR, and NOT.</span>
- <span style="color:#000ff;">Kyuss AND Green –</span> <span style="color:#000ff;"><i>lists results with keywords ‘Kyuss’ and ‘Green’</i></span>
- <span style="color:#000ff;">Zeppelin OR Floyd --</span> <span style="color:#000ff;"><i>lists results with keywords 'Zeppelin' or 'Floyd'</i></span>
- <span style="color:#000ff;">Metallica NOT Anger –</span> <span style="color:#000ff;"><i>lists all Metallica tracks except with the word ‘Anger’</i></span>
<span style="color:#000ff;">Instead of using AND or NOT you can also use a plus sign (+) or a minus sign (-).</span>
<span style="color:#000ff;">Further search options</span>
<span style="color:#000ff;">genre:</span> <span style="color:#000ff;"><i>Displays music in the genre matching keyword. See the</i></span> <a href="http://spreadsheets.google.com/pub?key=psnjFY3R2itsqjinSs9hkZw" rel="noopener" class="external-link" target="_blank" style="color:#e4afaff;"><i><u>list of all genres</u></i></a><span style="color:#000ff;"><i>, for example genre:blues, genre:"rock and roll".</i></span>
<span style="color:#000ff;">label:</span> <span style="color:#000ff;"><i>Displays music released by the label matching keyword.</i></span> 
<span style="color:#000ff;">isrc:</span> <span style="color:#000ff;"><i>Displays tracks matching ID number according to the International Standard Recording Code.</i></span> 
<span style="color:#000ff;">upc:</span> <span style="color:#000ff;"><i>Displays albums matching ID number according to the Universal Product Code</i></span><span style="color:#000ff;">.</span>
<span style="color:#000ff;">tag:new -</span> <span style="color:#000ff;"><i>Lists the most recently added albums. ‘Tag’ won’t return results for any other keywords.</i></span>
