# frmwrk_

> **Quick Access:** [[E - The Foundry]] | [[pixel glitch creative studio]]

## Project Overview

**frmwrk_** is a weekly Twitch stream brand built around DJ sets and creative work. Stream runs **Monays at 9PM PST**, covering UK garage, bass music, and hip-hop DJ sets alongside music production, creative tech, and gaming with Camille.

**Status:** Active — Phase 2 in progress
**Priority:** High
**Launch Target:** May 1, 2026
**Platform:** Twitch

The visual identity draws from **1987 OS aesthetics** — Windows 3.1 / early Macintosh UI conventions — combined with a muted analog color palette inspired by 1980s Japanese motorsport photography. The result is a functional-looking overlay system that feels archival, warm, and handmade rather than modern or polished.

All assets are built natively in **Photoshop, Illustrator, After Effects, and Premiere**. No HTML/CSS overlays.

## Visual System

### Color Palette

| Name | Hex | Role |
|---|---|---|
| Oil Green | `#6b6b2a` | Primary accent, title bars |
| Coral Red | `#d4614a` | Alerts, active states |
| Glaucous Blue | `#8fa3a8` | Window borders, secondary UI |
| Sulphur Yellow | `#c9b84c` | Highlights, ticker |
| Ash Violet | `#7a6472` | Inactive/muted elements |
| Persimmon | `#d4522a` | Warning states, hot accent |
| Celadon | `#9db89a` | VU meter active segments |
| Raw Umber | `#8b6340` | Wood-tone texture accents |
| Cream | `#f0ead6` | Text on dark backgrounds |
| Desktop BG | `#c8c0a8` | Canvas / desktop fill |

### Typography Stack

| Face | Use |
|---|---|
| **Press Start 2P** | Window title bars, UI labels |
| **VT323** | Data readouts, ticker text, session info |
| **Share Tech Mono** | Body text, chat feed, secondary labels |

### Window Chrome Rules

- **Bevel:** 1px light top/left, 1px dark bottom/right (classic Win 3.1 inset/outset)
- **Title bar:** Horizontal stripe pattern — alternating Oil Green / Cream, 2px height per stripe
- **Inactive title bar:** Stripe replaced by solid Ash Violet
- **Checker fill:** 2×2px checkerboard (Desktop BG + 10% darker) used for inactive regions
- **Window controls:** Minimize / maximize / close buttons as pixel-drawn squares in top-right of each title bar

## Stream Format

**Platform:** Twitch
**Schedule:** Sundays, 8:00 PM PST
**Stream length:** ~2 hours

### Segment Structure
1. **Starting Soon** — Animated loop with upcoming track info
2. **DJ Set** — Primary content, 60–90 min
3. **Creative Work / Tech / Gaming** — Secondary segment with Camille
4. **BRB** — Animated loop for breaks
5. **Ending** — Outro animation
