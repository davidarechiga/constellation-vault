# Storage & Garage

> Storage organized enough to find any item in 5 minutes. Seasonal swap within 2 weeks of season change.

**Frequency:** Seasonal (4x/year) + as-needed organization

---

## Seasonal Swap Checklist

Run at the start of each season (March, June, September, December):

- [ ] Pull out current season's items (clothes, gear, décor)
- [ ] Pack away previous season's items — labeled bins
- [ ] Check for items to donate or discard before storing
- [ ] Cyrus's seasonal clothing — size check, rotate out what's too small
- [ ] Baby gear seasonal review — what's needed now vs. can be stored?

## Organization Standard

- [ ] Can find any item in < 5 minutes (if not, reorganize that zone)
- [ ] All bins labeled — no mystery boxes
- [ ] Clear path through garage/storage area
- [ ] Nothing on the floor that belongs on a shelf
