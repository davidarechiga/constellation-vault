# Electronics & IT

> Wi-Fi works. Tech issues resolved within a week. Subscriptions audited.

**Frequency:** Monthly audit (1st of month) + issues addressed within 1 week

---

## Monthly Check

- [ ] Wi-Fi and router functioning normally?
- [ ] Any streaming/subscription services to cancel or review?
- [ ] Any devices running slow or needing updates?
- [ ] Any apps or services we're paying for but not using?

## Quarterly Subscription Audit

- [ ] List all active subscriptions and monthly cost
- [ ] Cancel anything unused for 2+ months
- [ ] Check for price increases on existing services
- [ ] Confirm shared family accounts are set up correctly (Camille has access)

## As-Needed

- [ ] Tech issue reported → resolve within 1 week
- [ ] New device setup → configure and onboard fully
- [ ] Password or account access issue → resolve same day if urgent
