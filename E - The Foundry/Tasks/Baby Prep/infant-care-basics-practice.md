# Infant Care Basics Practice

## Diapering
- [ ] Practice diaper changes on doll or stuffed animal
- [ ] Learn proper wiping technique (front to back for girls)
- [ ] Understand when to use diaper cream vs. ointment
- [ ] Practice one-handed diaper changes (common in real scenarios)

## Swaddling
- [x] Watch swaddling tutorial videos
- [x] Practice swaddling technique with receiving blankets
- [x] Learn swaddle blanket method
- [x] Practice hip-healthy swaddling (legs can bend and spread)

## Bathing
- [ ] Learn sponge bath technique for first weeks
- [ ] Understand umbilical cord care (keep dry until falls off)
- [ ] Practice safe bath hold positions
- [ ] Learn proper water temperature testing (elbow test)

## Feeding Basics
- [ ] Review breastfeeding positioning from class
- [ ] Practice bottle feeding technique (if bottle feeding)
- [ ] Learn proper burping positions and techniques
- [ ] Understand feeding cues vs. other cries

## Soothing & Comforting
- [ ] Learn the "5 S's" (Swaddle, Side/Stomach, Shush, Swing, Suck)
- [ ] Practice different holding positions
- [ ] Understand difference between cries (hungry, tired, uncomfortable, overstimulated)
- [ ] Practice skin-to-skin positioning

## Safety Basics
- [ ] Review safe sleep positioning (back to sleep)
- [ ] Understand when to call pediatrician vs. 911
- [ ] Learn signs of illness or distress in newborns
- [ ] Review CPR skills from certification class

**Why matters**: Practicing basic care skills before baby arrives builds confidence and reduces anxiety. Real babies are delicate and squirmy - practicing on dolls helps develop muscle memory for essential tasks.

**Done when**: Both David & Camille feel comfortable with diapering, swaddling, holding, feeding, and soothing techniques. Can demonstrate each skill competently.

## Common Newborn Characteristics

Be prepared for these normal newborn traits:
- **Fontanel** (soft spot on head) - Normal, will close over time
- **Frequent feeding** - 8-12 times per 24 hours
- **Umbilical cord stump** - Falls off in 1-3 weeks, keep dry
- **Newborn reflexes** - Startle reflex, rooting, sucking, grasping
- **Crying** - 2-3 hours per day is normal (peaks around 6 weeks)
- **Irregular sleep** - Newborns don't have day/night rhythm yet

## Pro Tips

- **Practice before sleep-deprived** - Build skills now while thinking clearly
- **Both partners practice** - Ensure both can handle all care tasks
- **Expect the unexpected** - Real babies move, cry, and pee during diaper changes
- **Ask for help** - Hospital nurses and doula (Verenice) will guide you through first attempts
